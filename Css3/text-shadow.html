CSS3 text-shadow Property

The text-shadow property adds shadow to text. text-shadow: h-shadow v-shadow blur-radius color; text-shadow: 2px 2px #ff0000; Output text-shadow: 2px 2px 8px #FF0000; Output text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF;
