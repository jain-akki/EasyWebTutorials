Form Elements


HTML Forms are required when you want to collect some data from the user. For example during user registration you would like to collect information such as name, email address etc. The different types of form elements that you can use to collect user data are:

The <input> Element: It is the most important form element. The <input> element has many variations, depending on the type attributes. Type attributes are:
• Single-line text input
• Password input
• Multi-line text input

Checkbox Attribute: Checkboxes are used when more than one option is required to be selected.

Radio Attribute: Radio buttons are used when out of many options, just one option is required to be selected.

File Upload Box: If you want to allow a user to upload a file to your web site, you will need to use a file upload box.

Button Element: There are various ways in HTML to create clickable buttons. Different button types are:

submit button - This creates a button that automatically submits a form.

reset button - This creates a button that automatically resets form elements to their initial values.

button - This creates a button that is used to trigger a client-side script when the user clicks that button.

Select Box Element: A select box, also called drop down box which provides various options in the form of drop down list, from where a user can select one or more options.

Attributes are: multiple, size, optgroup.
