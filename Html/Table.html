Table tag

The HTML tables allow you to arrange data like text, images, links, etc. into rows and columns. The HTML tables are created using the <table> tag in which <tr> tag is used to create table rows and <td> tag is used to create table data.

<caption> tag defines a table caption. It must be inserted immediately after the <table> tag.


Attributes Description
Align Left, Center, Right.
Background Specifies a background picture for the table.
Bgcolor Sets the background color.
Border Specify width of a border.
Bordercolor Specify color of a border.
Cellpadding cellpadding represents the distance between cell borders and the content within a cell.
Cellspacing The cellspacing attribute defines the width of the border.
Width, Height You can specify table width or height in terms of pixels or percentage.
Colspan To merge two or more columns into a single column.
Rowspan Rowspan To merge two or more rows.
