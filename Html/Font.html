Font Tag

To change the text font size, color and font face.

Attributes Description
Face To change font of the text.
Size To change text size. Size must be between 1 and 7. Default font size is 3.
Color To change text color.

Element Description
<b> ... </b> Or <strong> ... </strong> To Make Text Bold
<i> ... </i> Or <em> ... </em> To Make Text Italic.
<u> ... </u> Or <ins> ... </ins> To Make Text Underline.
<sup> ... </sup> To create text as Superscript.
<sub> ... </sub> To create text as Subcript.
<strike> ... </strike>, <s> ... </s> Or <del> ... </del> To Strike the Text.
<small> ... </small> Text appear in small size
<center> ... </center> To Make the Text in Center of the page.
& nbsp For Non Breaking Space
<q> ... </q> For Quotation marks
<blockquote> ... </blockquote> Usually use for indent the text
<br /> Break row: anything following it starts from the new line.
